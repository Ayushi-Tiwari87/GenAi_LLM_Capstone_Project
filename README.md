# Digital Well‑Being Capstone: Multi‑Layer Analytics and Decision Support

Digital Well‑Being Capstone is a multi-layer analytics and decision-support system that:
- Profiles users from screen time, sleep, stress, and activity data
- Predicts well‑being targets (e.g., happiness, stress)
- Clusters behavioral patterns into personas
- Retrieves evidence‑based guidance from literature and trusted sources
- Assembles personalized, seven‑day intervention plans
- Runs basic fairness checks across user subgroups

> Disclaimer: This project is for research and educational purposes only and is not a substitute for medical advice, diagnosis, or treatment.
